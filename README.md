# Hello OpenAI

這個專案是用來練習 OpenAI API 的使用，目前實現了以下功能：

- [x] 透過 GPT-3.5-Turbo 生成對話
- [x] 透過 DALL-E 生成圖片
- [ ] 透過 [outpainting](https://openai.com/blog/dall-e-introducing-outpainting) 將圖片擴展 (see [here](https://platform.openai.com/docs/guides/images/edits))
